# ngmh's cheatsheet repository

some cheatsheets i made for nus modules (mostly math / cs coded), no point gatekeeping (since i spent all this effort typing them out anyway)

how to use: open the `pdfs` folder and find your desired cheatsheet (or use the handy table below)

i do not take any responsibility for any loss of gpa, module content can change over time and there could be errors here and there (feel free to patch them)

template borrowed from [here](https://github.com/jovyntls/cheatsheets), mostly for the multicols tbh (i love overleaf)

i think plain cheatsheets are boring so `funny` has cheatsheets with funny backgrounds to keep me entertained in the mpsh (yes me bored)

if you found this repository helpful please drop a star and tell your friends :)

MY CHEATSHEETS ARE FREE TO USE! STOP UPLOADING IT TO STUDOCU (PLEASE DON'T PAY FOR THEM!!!) never thought i would be saying this
