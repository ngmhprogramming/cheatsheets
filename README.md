# ngmh's cheatsheet repository

some cheatsheets i made for nus modules (mostly math / cs coded)

how to use: open the `pdfs` folder and find your desired cheatsheet

btw i do not take any responsibility for any loss of gpa, module content can change over time and there could be errors here and there

template borrowed from [here](https://github.com/jovyntls/cheatsheets), mostly for the multicols tbh

i think plain cheatsheets are boring so `funny` has cheatsheets with funny backgrounds to keep me entertained in the mpsh

if you found this repository helpful please drop a star and tell your friends :)
